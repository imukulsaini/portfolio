Difference between innerHtml and innerText in JAVASCRIPT

January 2021 2 min Read .by Mukul

In this blog, i will discuss about what is the Difference Between innerHtml and innerText .


Let's take an example here is code :

For innerText

document.getElementById("headingExample").innerText = 'print innerText';
OutPut :
It will only Print the text from the reference H1.

For innerHtml

document.getElementById("headingExample").innerText = 'print innerHtml';
OutPut :
It will Print the whole Html from the reference H1.
